# BootArt
## A Graphical Operating System

build:
```
$ make boot_minimal
$ make boot_keyboard_input
$ make boot_heart
$ make boot_read_disk
$ make boot_protected_mode
```

run:
```
$ sh run.sh minimal
$ sh run.sh keyboard_input
$ sh run.sh heart
$ sh run.sh read_disk_
$ sh run.sh protected_mode
```
